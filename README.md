## Description

This is a console application which downloads images of cards from fandom websites for the Villainous board game.

## Download & Run

### Download the executable
Get the latest release here: [Download here](https://dotnet.microsoft.com/en-us/download/dotnet/6.0/runtime)

⚠️ Requirements:
- Windows 64-bit
- .NET 6 Runtime: [Download here](https://dotnet.microsoft.com/en-us/download/dotnet/6.0/runtime)

### Run
Start Villainous.exe
<br/>
<img width="545" height="158" alt="image" src="https://github.com/user-attachments/assets/320b99e5-36f7-4f8c-b7bf-91e9762212b4" />

Choose the type of Villainous game.
For Disney, Marvel, Star wars or Anime type d,m,s,a and hit Enter. For Disney Homebrew, Marvel Homebrew, type dh or mh.
Type the name of the villain. Make sure to type it correctly as it is written in the website. The console app will start downloading the images.

### Result
You can find them in the directory of the exe in a folder named after the villain. You can see images for the realm and for the card backs.
Folders:
 - folders named 1,2,3,etc. mean that the card is repeated that number of times.
 - Special folder is for special cards that don't follow the traditional HTML structure.
 - Guide is for images for the pages of the villain guide
